# Domain Availability Checker Instructions

## Quick Domain Check Commands

You can use these commands to quickly check if domains are available:

### Windows Commands:
```cmd
nslookup rank.com
nslookup pulse.com
nslookup rise.com
```

### Alternative: Online Tools
1. **GoDaddy Domain Search**: Go to godaddy.com and use their domain search tool
2. **Namecheap**: Visit namecheap.com and check availability
3. **Google Domains**: Use domains.google.com
4. **Whois Lookup**: Visit whois.com
